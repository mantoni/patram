# v0 Conventions Proposal

- Kind: convention
- Status: active

## Config

- Config file: `.patram.json`.
- Config format: JSON.
- Paths: repo-relative.
- Path separator: `/`.
- Include patterns: glob strings.

```json
{
  "include": ["docs/**/*.md"],
  "queries": {
    "pending": {
      "where": "kind=task and status=pending"
    },
    "blocked": {
      "where": "kind=task and status=blocked"
    },
    "accepted-decisions": {
      "where": "kind=decision and status=accepted"
    }
  }
}
```

## Parsing

- Markdown first.
- Title: first line.
- Heading title: strip leading `#`.
- Link claim type: `markdown.link`.
- Directive claim type: `directive`.
- Directive names normalize to `lower_snake_case`.

## Identity

- Document node id: `doc:<path>`.
- Non-document node id: `<kind>:<key>`.
- Claim id: `claim:<document_id>:<ordinal>`.

## Graph

- Built-in kind: `document`.
- Built-in relation: `links_to`.
- v0 graph layers: `schema`, `claims`, `graph`.
- Graph nodes are concrete objects.
- Kinds and relations live in schema.
- Query target: graph nodes.

## CLI

- Root command: `patram`.
- Config discovery: current working directory first.
- Default target for `check`: current working directory.
- `query --where` runs an ad hoc query.
- `query <name>` runs a stored query.
- `queries` lists stored queries.
- `show` requires one file path.

## Output

- Human-readable text first.
- No JSON CLI output in v0.
- Diagnostic format: `<path>:<line>:<column> <level> <code> <message>`.
- Query result format: `<id> <kind> <title-or-label>`.
- Query list format: `<name> <where>`.
- `show` output order:
  - original source
  - resolved summary

## Query

- Query command name: `query`.
- Stored queries live in `.patram.json`.
- Stored query lookup is exact by name.
- v0 query language is small.
- v0 query language filters nodes only.
- v0 query language supports relation existence checks.
- No traversal in v0.

```txt
kind=task and status=pending
kind=task and not blocked_by:*
kind=decision and status=accepted
path^=docs/roadmap/
title~query
```
